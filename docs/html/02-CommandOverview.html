Updating the multiviewer files

(~ = ALT+N on Mac

  1. cd ~/Projects/annotation (press tab)
  2. git pull origin ibt_release
    1. enter github user name
    2. enter github password

Steps to run the multiviewer


  1. Open Finder and make sure you can access juliette
    1. CMD + Space ⇒ Finder or open finder from the doc at the bottom of the desktop
  2. Open Konsole via spotlight and open konsole:
    1. CMD + Space ⇒ Terminal
  3. Navigate to multiviewer folder:
    1. you can within the terminal press tab to use auto-completion
    2. on mac you can do ~ with ALT + N
    3. cd ~/Projects/anno (press tab) /src
  4. activate the multiviewer environment
    1. source activate multiviewer
  5. you should now see (multiviewer) before your path
  6. Start multiviewer IBT version with p_num and subsequence:
    1. python run_multiviewer_IBT.py --p-num 001 --subsequence bg-01


Navigating the sequences


  1. Press ö and ä to navigate front and back on the sequences.
  2. Press 1, 2, 3, 4, 5 to navigate frames18, frames54, frames90, frames126, frames 162.
    1. Press 1 to select the sequence frames18 for navigation. Then again, press ö and ä to navigate front and back on the sequences.
    2. press 2 to selection the sequence frames54 for navigation. Then again, press ö and ä to navigate front and back on the sequences.
    3. ...
  3. Press w to navigate all sequences at once.


Adding Annotations


  1. The command line constantly shows you which leg (left or right) you selected
    2. Press l or r to switch between left and right leg targets
  2. The command line also constantly shows you all the annotations you created so far
  3. When navigating the sequences, you can press
    1. a to annotate an IN_THE_AIR case
    2. g to annotate an ON_THE_GROUND case
    3. n to annotate a NOT_IN_FRAME case
    4. d to dete the last annotation in the list

  • Press g ö a ö to annotate the current frame as ON_THE_GROUND, go 1 frame forward, annotate as IN_THE_AIR, go another frame forward

Saving Offsets


  1. Press j to save the current offset between the sequences.
    1. You will get a command output with the basename and the filename of the file to save.
  2. If you already have an offsets file, you have the following 3 options
    1. Press d to delete the offsets file and NOT save the offsets file
    2. Press s to overwrite the current offsets file with new offsets file.
    3. Press any key to continue without any action.

Saving Annotations


  1. Press c to save the annotations to a csv file
    1. You can verify the location of the saved through the command line output


Exiting the Multiviewer


  1. Press ESC
    1. You will be asked to confirm by pressing ESC again
